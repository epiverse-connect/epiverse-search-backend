# Run the Shiny Application

```r
run_app(...)
```

## Arguments

- `...`: Unused arguments.

## Description

Runs the 'apisensr' Shiny application. This function does not return; interrupt R to stop the application (usually by pressing Ctrl+C or Esc).

## Examples

```r
## Not run:

run_app()
## End(Not run)
```
